📖 가계부 사용법
💸 수입 / 지출 기록
  • 상단 입력폼에서 날짜 · 유형 · 카테고리 · 금액 · 메모를 입력 후 추가하면 기록돼요.
  • 유형을 수입으로 선택하면 이달 총수입에 합산되고, 예산이 자동 계산돼요.
  • 기록된 내역을 클릭하면 수정 모드로 진입해요. 수정 후 저장하거나 취소 가능해요.
📊 월수입 배분 잔액
  • 배분 잔액 = 이달 총 수입 그 자체예요.
  • 예산 진행바는 생활비(배분제외 항목 제외한 실제 지출)가 총 수입 대비해서 어느정도인지 보여줘요.
🔖 월수입 배분 잔액에서 제외
  • 지출 입력 시 "월수입 배분 잔액에서 제외" 체크박스를 체크하면, 해당 항목은 생활비가 아닌 미리 빼둔 돈으로 처리돼요.
  • 예) 비상금 사용 등
  • 이 항목들은 목록에 "배분제외" 뱃지로 표시되고, 차트에서 생활비 누적에는 포함되지 않아요.
  • 단, 총 지출 누적 선에는 포함돼요.
📈 투자 기록
  • 투자 매수/매도는 상단 주식/투자 자산 카드를 클릭해서 계좌 관리 창에서 직접 수정해요.
  • 투자 지출을 가계부에 기록할 때는 카테고리 투자로 일반 지출로 기록하고, "월수입 배분 잔액에서 제외"에 체크해두면 생활비에서 빠져요.
🔄 고정지출 관리
  • 매달 반복되는 지출(통신비·넷플릭스 등)을 고정지출 관리에 등록해두면, 매달 해당 날짜에 자동으로 기록 여부를 물어봐요.
  • 설명, 금액, 매월 결제일을 입력해서 등록하세요.
📅 달력 / 차트
  • 달력에서 날짜별 +수입 / -지출 금액을 한눈에 볼 수 있어요.
  • 차트는 생활비 누적(빨강)과 총 지출 누적(노랑)을 일별로 보여주고, 초록 점선이 배분 잔액 기준선이에요.
  • 현재 날짜 이후 구간은 데이터가 없어서 점이 표시되지 않아요 (정상이에요).
🏦 자산 관리
  • 주식/투자 자산, 예적금 자산 카드를 클릭하면 각 계좌를 직접 관리할 수 있어요.
  • 시세 업데이트 버튼으로 주식·금 시세를 실시간 반영해요.
💾 백업 / 복원
  • 백업 저장으로 JSON 파일에 모든 데이터를 저장하고, 백업 불러오기로 복원할 수 있어요.
  • 엑셀 저장으로 거래 내역을 CSV로 내보낼 수 있어요.
🔄 고정 지출 관리

매달 반복되는 지출(월세, 구독료 등)을 등록하세요.
